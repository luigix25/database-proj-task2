  MONGODB REPLICA
===================

Start MongoDB daemon with Replicas
$ mongod --replSet BSSReplica

Prepare rsconf
rsconf = {
    _id: "BSSReplica",
    members: [
        { _id: 0, host: "localhost:27017"},
        { _id: 1, host: "localhost:27018"}
    ]
}

Start replica
    rs.initiate(rsconf)

Show configuration
    rs.conf()

Show status
    rs.status()

--- Add new server to set
    rs.add(
        {
            host: "mongodb3.example.net:27017",
            priority: 0,
            votes: 0
        }
    )
--- Remove server from set
rs.remove("mongod3.example.net:27017")

===== Dump of a MongoDB Database =====
$ mongodump -d database -h example.org -u username -p password -o backupdir --authenticationDatabase=admin

Example:
$ mongodump -d ducange -h dutcher.robinodds.it -u root -p prova -o /mnt --authenticationDatabase=admin

$ mongorestore -d database backupdir
